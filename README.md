# Debounced Controlled Component

A state container to debounce a controlled component.

```javascript
<DebounceControlledComponent
  time={300}
  value={initialValue}
  onChange={e => e.target.value}
>{ (value, debouncedOnChange) =>
  <input
    value={value}
    onChange={debouncedOnChange}
  />
}</DebounceControlledComponent>
```
